Every Picture Project Development

I'm hoping to create an interaction where a user will hover over one of the Jellycats and it will slightly grow. Then, when they click on it, the Jellycat will grow a lot and move to the right side, allowing room for the overlay with text to pop up. I'm hoping this effect will make it feel like the user is selecting a subject from a set of choices in hopes of learning more about it. I plan on including text that gives information about each Jellycat, the story of how it joined my collection, and its significance to me.
